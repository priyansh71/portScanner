## Customized Port scanner using sockets.

To run the file,
* ```python3 scannerExt.py -flagExt``` (more features, but slow)
* ```python3 scanner.py -flag``` (less features, but faster)

where:
  * flagExt = ```-ip -pR -pA -op -proto``` <br />
  * flag = ```-ip -pR -op```

For help, do
```python3 [filename] -h```
