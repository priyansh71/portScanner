## Customized Port scanner using sockets.

To run the file,
```python3 scannerExt.py -flagExt``` (more featured, but slow)
```python3 scanner.py -flag``` (less featured, but faster)

where:
  flagExt = ```-ip -pR -pA -op -proto```
  flag = ```-ip -pR -op```

For help, do
```python3 [filename] -h```
